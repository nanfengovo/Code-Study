
| 序号  |   专用名词   |                   全称                   |                                                                      解释                                                                       |
| --- | :------: | :------------------------------------: | :-------------------------------------------------------------------------------------------------------------------------------------------: |
| 1   |  Token   |       狭义指**令牌**，在大模型领域为**语义单元**        |         1. 在计算机安全领域：是身份验证的凭证，用于替代密码，实现安全访问；<br><br>2. 在大语言模型（LLM）领域：是文本的最小语义单元，一个汉字、一个英文单词或标点都可能被计为一个 Token，模型的输入输出上限通常以 Token 数量衡量。          |
| 2   |   MCP    |    Model Context Protocol（模型上下文协议）     |                          一种用于连接大语言模型与外部工具 / 系统的通信协议，核心作用是让模型能够理解并调用外部资源（如数据库、API、硬件设备），扩展模型的能力边界，常与 AI Agent 架构结合使用。                          |
| 3   | AI Agent | Artificial Intelligence Agent（人工智能智能体） |                一种能够**自主感知环境、制定决策、执行任务**的人工智能系统。它具备目标导向能力，可独立完成复杂任务（如自动写代码、数据分析、多步骤规划），还能根据任务反馈调整策略，典型结构包含感知模块、规划模块、执行模块和记忆模块。                 |
| 4   |   RAG    | Retrieval-Augmented Generation（检索增强生成） | 一种大语言模型优化技术，核心流程是**先检索、再生成**：<br><br>1. 检索：从外部知识库（如文档、数据库）中查找与用户问题相关的信息；<br><br>2. 生成：模型结合检索到的精准信息和自身知识进行回答，既解决了模型 “知识过期” 的问题，又提升了回答的准确性和可信度。 |
